## Overview

A `DatePicker` is used to select a date. The input is modelled around the
[`Combobox`](../?path=/docs/inputs-combobox--docs) ARIA pattern with a `dialog` Calendar that can be navigated via
keyboard arrows

## Variants

### Basic Usage

By default, the DatePicker is localised to the user's browser locale whilst the calendar's initial date is set to
`today`.

<Canvas of={DatePickerStories.Base} />

### Min/Max date

It's also possible to cap the date range that can be selected. If you don't supply an intial date when using the min/max
date props and `today` is outside of the date range it automatically sets to the minimum date.

<Canvas of={DatePickerStories.MinMaxDate} />

### Enforcing a locale

If you want to enforce a locale, you can pass it as a prop. This will override the user's browser locale.
