## Overview

A `TimePicker` is used to select a specific time. The input is modelled around the
[`Combobox`](https://github.com/strapi/design-system/tree/main/packages/strapi-design-system/src/Combobox) ARIA pattern
with a `listbox` option menu that can be navigated with your keyboard. The returned value is a `string` _not_ a `Date`.

[View source](https://github.com/strapi/design-system/tree/main/packages/strapi-design-system/src/TimePicker)
