# DismissibleLayer

`DismissibleLayer` is a internal utility component re-exported from the `@radix-ui/dismissable-layer` package. It's
typically used within a `Modal` or a `Popover` & in conjunction with `FocusTrap`.

<ViewSource path="utilities/DismissibleLayer" />

## Usage

```js
import { DismissibleLayer } from '@strapi/design-system';
```
