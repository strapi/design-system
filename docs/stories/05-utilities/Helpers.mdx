# Helpers

Several utility functions and constants are exported from the design system for use in your projects. Use them to ensure consistency of styling and behavior with the design system (e.g. theme tokens, responsive values, keyboard keys, reduce opacity of a hex color code).
<ViewSource path="helpers" />

## Import

```tsx
import {
  setOpacity,
  handleResponsiveValues,
  KeyboardKeys,
  extractStyleFromTheme,
  type ResponsiveProps,
  type Breakpoint,
} from '@strapi/design-system';
```

---

## setOpacity

Returns a hex color string with the given opacity (alpha) applied. Useful for overlays, disabled states, or any semi-transparent UI that should stay on brand.

### Signature

```ts
setOpacity(hex: string, alpha: number): string
```

### Parameters

- **hex** (string) – A hex color (e.g. `#000000` or `#abc`)
- **alpha** (number) – Opacity from 0 to 1 (e.g. `0.5` for 50% opacity)

### Example

```tsx
import { setOpacity } from '@strapi/design-system';

// In a styled-component or CSS-in-JS
const Overlay = styled.div`
  background: ${({ theme }) => setOpacity(theme.colors.neutral800, 0.2)};
`;
```

## handleResponsiveValues

Takes an object of responsive props (spacing, layout, typography, etc.) and the theme, and returns a string of CSS (including media queries) that you can inject into styled-components or any CSS-in-JS solution. Supports breakpoints: `initial`, `small`, `medium`, `large`.

### Signature

```ts
handleResponsiveValues(values: ResponsiveProps, theme: DefaultTheme): string
```

### Parameters

- **values** (ResponsiveProps) – Object of CSS-related props; values can be theme tokens or responsive objects (e.g. initial: 2, large: 4)
- **theme** (DefaultTheme) – Strapi theme (e.g. from `useTheme()` or `props.theme` in styled-components)

### Responsive value shape

A property can be a single value (used as the initial value) or an object keyed by breakpoint:

```ts
// Single value
padding: 2

// Responsive object
padding: { initial: 1, medium: 2, large: 4 }
```

### Exported types

- **ResponsiveProps** – Props object accepted by `handleResponsiveValues` (spacing, layout, typography, etc.)
- **Breakpoint** – `'initial' | 'small' | 'medium' | 'large'`
- **ResponsiveProperty&lt;T&gt;** – A value or object keyed by breakpoint
- **ResponsiveThemeProperty** – Theme-aware responsive property type

### Example

```tsx
import { handleResponsiveValues } from '@strapi/design-system';
import styled from 'styled-components';

const Card = styled.div`
  ${({ theme }) =>
    handleResponsiveValues(
      {
        padding: { initial: 2, medium: 4, large: 6 },
        marginBottom: 4,
      },
      theme,
    )}
`;
```

## KeyboardKeys

An object of common keyboard key values. Use it when handling keyboard events so you compare against stable string values instead of magic strings.

### Keys

- **DOWN** – `'ArrowDown'`
- **UP** – `'ArrowUp'`
- **RIGHT** – `'ArrowRight'`
- **LEFT** – `'ArrowLeft'`
- **ESCAPE** – `'Escape'`
- **ENTER** – `'Enter'`
- **SPACE** – `' '`
- **TAB** – `'Tab'`
- **END** – `'End'`
- **HOME** – `'Home'`
- **DELETE** – `'Delete'`
- **PAGE_UP** – `'PageUp'`
- **PAGE_DOWN** – `'PageDown'`
- **BACKSPACE** – `'Backspace'`
- **CLEAR** – `'Clear'`

### Example

```tsx
import { KeyboardKeys } from '@strapi/design-system';

const handleKeyDown = (event: React.KeyboardEvent) => {
  if (event.key === KeyboardKeys.ESCAPE) {
    onClose();
  }
  if (event.key === KeyboardKeys.ENTER || event.key === KeyboardKeys.SPACE) {
    event.preventDefault();
    onSubmit();
  }
};
```

## extractStyleFromTheme

Looks up a value from a theme section (e.g. `theme.colors`, `theme.spaces`) by key and returns the resolved style value, or the default if the key is missing or invalid. Use it when building custom styled components that should respect theme tokens.

### Signature

```ts
extractStyleFromTheme<TKey, TSection>(
  themeSection: TSection | null | undefined,
  key: string | number | symbol | undefined,
  defaultValue: any
): string | number
```

### Parameters

- **themeSection** (object) – A slice of the theme (e.g. `theme.colors`, `theme.spaces`, `theme.borderRadius`)
- **key** (string, number or symbol) – The token key to look up (e.g. `'neutral800'`, `2`)
- **defaultValue** (any) – Value to return when the key is not found in the section

### Example

```tsx
import { extractStyleFromTheme } from '@strapi/design-system';

const Badge = styled.span`
  background: ${({ theme }) =>
    extractStyleFromTheme(theme.colors, 'primary600', theme.colors.primary600)};
  padding: ${({ theme }) =>
    extractStyleFromTheme(theme.spaces, 1, '4px')};
  border-radius: ${({ theme }) =>
    extractStyleFromTheme(theme.borderRadius, 1, '4px')};
`;
```
