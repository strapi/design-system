# Responsive

This documentation explains how responsive values work across the design system.

## How It Works

A responsive prop accepts either:

- A single value (applied to the initial breakpoint)
- An object of breakpoint keys to values

Breakpoints are defined in the theme as:

- `initial`: 0px
- `small`: 520px
- `medium`: 768px
- `large`: 1080px

## Single Value

```tsx
<Box padding={2} />
```

## Responsive Object

```tsx
<Box padding={{ initial: 2, medium: 4, large: 6 }} />
```

## Responsive Array

```tsx
<Box padding={[2, 4, 6]} />
```

Arrays map to breakpoints in order: `initial`, `small`, `medium`, `large`.
