## Overview

A composite component containing everything you need to make fully accessibility form inputs using either the provided
`input` element or your own custom input.

## Usage

```js
import { Field, useField } from '@strapi/design-system';
```

If you're using your own custom input not based off `Field.Input` then utilise the `useField` hook to gain access to the
following props:

```ts
interface Field {
  /**
   * @default false
   */
  error?: string | boolean;
  /**
   * @default null
   */
  hint?: React.ReactNode;
  id?: string;
  labelNode?: HTMLLabelElement;
  name?: string;
  /**
   * @default false
   */
  required?: boolean;
}
```
