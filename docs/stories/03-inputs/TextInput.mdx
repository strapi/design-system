## Overview

TextInputs are inputs that allow users to add or edit text. They support several text formats including numbers. If a
single line is not enough for the amount of text required, use [`TextArea`](../?path=/docs/inputs-textarea--docs)
component.

### Passwords

Password TextInputs are specific. They offer the possibility to have unreadable content. By clicking on the eye icon on
the right side, it is possible to actually read the password.
