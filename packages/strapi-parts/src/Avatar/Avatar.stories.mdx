# Avatar

This is the doc of the `Avatar` component

## Avatar

Avatars are used to show a thumbnail of an asset or an individual. 

**Best practices**

- Use the thumbnail preview feature only when the avatar might not be enough.
- Use avatar in small areas such as tables.
- In a form we should rather use the Card component.

## Usage

The avatar has a feature to display a preview of its thumbnail.
