# FocusTrap

The FocusTrap restricts keyboard navigation to a certain zone of the screen. For instance, it can be used in a modal to limit tab navigation to a specific context.

## Imports

```js
import FocusTrap from '@strapi/icons/FocusTrap';
```

## Usage

To use it, wrap your code into a `<FocusTrap onEscape={onClose}>` block.
