# Pagination

The Pagination component is used to navigate multipage content.

[View source](https://github.com/strapi/parts/tree/develop/packages/strapi-parts/src/Pagination)

## Imports

```js
import { Dots, NextLink, PageLink, Pagination, PreviousLink } from '@strapi/parts/Pagination';
```

## Usage

<Canvas>
  <Story name="base">
    <Pagination activePage={1} pageCount={26}>
      <PreviousLink to="/1">Go to previous page</PreviousLink>
      <PageLink number={1} to="/1">
        Go to page 1
      </PageLink>
      <PageLink number={2} to="/2">
        Go to page 2
      </PageLink>
      <Dots>And 23 other links</Dots>
      <PageLink number={25} to="/25">
        Go to page 3
      </PageLink>
      <PageLink number={26} to="/26">
        Go to page 26
      </PageLink>
      <NextLink to="/3">Go to next page</NextLink>
    </Pagination>
  </Story>
</Canvas>
