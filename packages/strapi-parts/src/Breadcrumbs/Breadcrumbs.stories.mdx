# Breadcrumbs

This is the doc of the `Breadcrumbs` component

## Breadcrumbs

Breadcrumbs are a visual information that inform users of their navigation. 

**Best practices**

- Use breadcrumbs whenever a long path to achieve an action is required.
- Use breadcrumbs in modal's headers.
- Using breadcrumbs in a page or a form is *not* a consistent experience.

## Usage

Breadcrumbs are visual information only and cannot be navigated. They are mostly part of modals' headers.
