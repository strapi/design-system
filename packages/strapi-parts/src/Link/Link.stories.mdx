# Link

Links are used to send users to another page or an anchor. They are used for external navigation.

**Best practices**

- Use TextButton component if an action needs to be triggered within the admin panel (ex: Opening a modal ).
- Clicking on a Link most of the time opens a new tab.
- Links should always be used with an icon. If it is an external link, use the ExternalLink icon accordingly.
- Links text should be concise and describe their purpose. Users need to know exactly what they are about to do.
- Use capital letters.


[View source](https://github.com/strapi/parts/tree/develop/packages/strapi-parts/src/Link)

```js
import { Link } from '@strapi/parts/Link';
```

## Usage

Links are used for external navigation. Moreover, they should be associated with an icon to indicate its external navigation purpose.

### Base

The simplest representation of the Link component.

### Disabled link

Depending on the permissions a user have or the status of an action, a link can be unreachable or unavailable yet.
