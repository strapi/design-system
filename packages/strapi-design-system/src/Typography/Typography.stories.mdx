## Imports


### Ellipsis

By using the prop `ellipsis` it is possible to limit the text to one line and to signal to the user there is more to it.

<Canvas>
  <Story name="ellipsis">
    <Box maxWidth="150px" borderStyle="solid" borderWidth="1px" borderColor="neutral200">
      <Typography ellipsis>Loong text, that will be cut off</Typography>
    </Box>
  </Story>
</Canvas>
