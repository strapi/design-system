# RawTable

This is the doc of the `RawTable` component

[View source](https://github.com/strapi/design-system/tree/main/packages/strapi-design-system/src/RawTable)

## Imports

```js
import { RawTable, RawTh, RawTd, RawTr, RawThead, RawTbody } from '@strapi/design-system';
```
