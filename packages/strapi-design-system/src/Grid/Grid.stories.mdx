# Grid

This is the doc of the `Grid` component

## Grid

The component allows to create a Grid on the screen using the CSS grid feature. This component extends the `<Box />` one and can receive the same props.

<Canvas>

    <Grid
      areas={['first second third', 'last  last last']}
      cols="1fr 1fr 1fr"
      rows="1fr 1fr"
      background="primary700"
      gap={4}
      margin={8}
      padding={4}
    >
      <Box area="first" color="neutral0">
        First
      </Box>
      <Box area="second" background="neutral100">
        Second
      </Box>
      <Box area="third" background="neutral100">
        Third
      </Box>
