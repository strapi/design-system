# InputJSON

Input JSON takes a json string and shows your data in clear editable view.

[View source](https://github.com/strapi/design-system/tree/main/packages/strapi-design-system/src/InputJSON)

## Imports

```js
import { InputJSON } from '@strapi/design-system';
```
