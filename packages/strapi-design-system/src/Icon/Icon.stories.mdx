# Icon

The `Icon` component is used to customize Strapi icons.

## Usage

<Canvas withSource="open">
  <Story name="base">
    <Stack spacing={3} horizontal>
      <Icon width={`${25 / 16}rem`} height={`${25 / 16}rem`} color="secondary500" as={Pencil} />
      <Icon width={`${15 / 16}rem`} height={`${15 / 16}rem`} color="alternative500" as={Play} />
      <Icon width={`${30 / 16}rem`} height={`${30 / 16}rem`} color="danger500" as={Trash} />
      <Icon as={Plus} />
    </Stack>
  </Story>
</Canvas>

## Props

The Icon component wraps all its children in the
(Box)[https://design-system-git-main-strapijs.vercel.app/?path=/docs/design-system-technical-components-box--base]
component, so you can pass all Box props to change its style.
